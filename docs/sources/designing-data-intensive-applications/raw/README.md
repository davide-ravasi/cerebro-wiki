# DDIA — raw chapter notes

Appunti di lavoro mentre leggi *Designing Data-Intensive Applications* — **in italiano**, cattura veloce (anche da **appunti a mano** → scan/foto).

## Workflow

1. Leggi + appunti su carta (o bullet veloci).
2. Passa scan o note → raw markdown (schema sotto).
3. Rivedi → promuovi source + concept in **inglese** (`../ch-NN-*.md`, `docs/concepts/`).
4. **`## Book club`** in fondo al raw — riassunto breve in **inglese**, da incollare su **Discord** (book club). Export dedicato: [`../book-club/`](../book-club/) (solo testo paste-ready).

Schema: [`docs/templates/raw-chapter-note.md`](../../../templates/raw-chapter-note.md)

| Esempio | Contenuto |
|---------|-----------|
| [`chapter-8.md`](chapter-8.md) | Raw completo + book club |
| [`chapter-4.md`](chapter-4.md) | Book club (compatibilità, formati binari) |

Files: `chapter-1.md` … `chapter-8.md` (e altri man mano).

## Book club (Discord)

- Sezione finale: `## Book club`
- Lingua: **English** ( anche dentro raw IT )
- Tono: “Here’s my take…”, takeaway personali, incollabile in Discord
- Chiedi **“book club”** in chat per generare o aggiornare solo quella sezione
