## Naming

- Group by module folder, e.g. `mongodb-document-model/`.

## Promote workflow

1. Finish or review a lesson in `raw/`.
2. Optional: add a formal `source` file in `../` with frontmatter (`type: source`, tags `mongodb`, `course`).
3. Link reusable ideas to `docs/concepts/` (e.g. embedding vs referencing → can become a small concept or stay in source only).
